# 🩸 HemoScan — Non-Invasive Anemia Screening

> **No Needle. No Lab. No Wait.**

HemoScan is a real-time health monitoring system that estimates **hemoglobin (Hb)** and detects **anemia** using **PPG signals** and an **ESP32-based sensor setup**.

---

## 🚀 Features

* ❤️ Heart Rate, 💧 SpO₂, 🧬 Hemoglobin (Hb)
* 📡 ESP32 WebSocket integration
* 📉 Live PPG waveform (Red, IR, Amber)
* 🩸 Anemia classification (Normal → Severe)
* 🚨 Emergency alerts
* 🤖 AI health advice
* 📄 PDF report download
* 📈 Hb trend tracking
* 🌐 English + Hindi UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Libraries:** Chart.js, jsPDF
* **Hardware:** ESP32 + Optical Sensors
* **Communication:** WebSocket

---

## ▶️ Usage

1. Open `index.html`
2. Connect ESP32 (`192.168.4.1`)
3. Click **Start Scan**
4. View results / download report

👉 Use **Demo Mode** if no hardware is connected.
